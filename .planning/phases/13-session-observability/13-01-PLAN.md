      - "error": timestamp, sessionID, errorType, message, model (optional), retryable
      - "decision": timestamp, sessionID, phase, agent, decision, rationale, confidence

    - ToolMetric interface: tool, invocations, totalDurationMs, successCount, failureCount

    - sessionLogSchema Zod schema for SessionLog
    - createEmptyTokenAggregate() helper returning zeroed aggregate
    - All types use `readonly` arrays and fields per project immutability pattern

    Create `src/observability/event-store.ts` with:
    - SessionEvents internal interface: sessionID, startedAt, events array, tokens (TokenAggregate), toolMetrics (Map<string, ToolMetric>), currentPhase (string | null)
    - SessionEventStore class with private `sessions: Map<string, SessionEvents>`
    - initSession(sessionID): creates entry with empty events, zeroed tokens, empty tool metrics
    - appendEvent(sessionID, event): pushes to events array; no-op if session unknown; drops if at maxEventsPerSession cap (default 500)
    - accumulateTokens(sessionID, tokens): immutably updates token aggregate using spread pattern
    - recordToolExecution(sessionID, tool, durationMs, success): upserts tool metric entry
    - setCurrentPhase(sessionID, phase): sets current phase for attribution
    - getCurrentPhase(sessionID): returns current phase or null
    - getSession(sessionID): returns session data or undefined
    - flush(sessionID): returns SessionEvents and deletes from map
    - Constructor accepts optional maxEventsPerSession parameter

    Create `src/observability/retention.ts` with:
    - pruneOldLogs(logsDir, retentionDays): reads directory, stats each .json file, unlinks if mtime older than cutoff, returns prune count
    - Only prunes .json files (summaries are generated on-demand, not persisted as .md files)
    - Uses isEnoentError for graceful handling of missing directory
    - Pattern follows src/orchestrator/lesson-memory.ts prune-on-load approach

    Create `src/observability/index.ts` barrel exporting all public types and classes.
