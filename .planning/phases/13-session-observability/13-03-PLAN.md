      - Returns null on ENOENT or validation failure
    - `readLatestSessionLog(logsDir?: string): Promise<SessionLog | null>`:
      - Lists all .json files, stats for mtime, returns the newest
    - `listSessionLogs(logsDir?: string): Promise<readonly SessionLogEntry[]>`:
      - Reads directory, parses each .json file header (sessionID, startedAt, endedAt, token summary)
      - Returns sorted by startedAt descending
      - `SessionLogEntry` type: { sessionID, startedAt, endedAt, totalCost, messageCount, eventCount }
    - `searchEvents(sessionLog: SessionLog, filters: { type?: string; after?: string; before?: string }): readonly ObservabilityEvent[]`:
