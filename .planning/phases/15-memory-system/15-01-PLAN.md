Create `src/memory/database.ts`:
- Import `Database` from `"bun:sqlite"` and `mkdirSync` from `"node:fs"`, `join` from `"node:path"`
- Import `getGlobalConfigDir` from `"../utils/paths"`, `MEMORY_DIR`, `DB_FILE` from `"./constants"`
- Module-level `let db: Database | null = null`
- `getMemoryDb(dbPath?: string): Database` — if db exists, return it. Otherwise: if no dbPath provided, compute from `join(getGlobalConfigDir(), MEMORY_DIR, DB_FILE)` and `mkdirSync` the parent. Create `new Database(dbPath)`, run `PRAGMA journal_mode=WAL`, `PRAGMA foreign_keys=ON`, call `runMigrations(db)`, assign to module var, return.
- `closeMemoryDb(): void` — if db, call db.close(), set db = null.
- `runMigrations(db: Database): void` — run all CREATE TABLE IF NOT EXISTS statements:
  - `projects` table: id TEXT PRIMARY KEY, path TEXT NOT NULL UNIQUE, name TEXT NOT NULL, last_updated TEXT NOT NULL
  - `observations` table: id INTEGER PRIMARY KEY, project_id TEXT, session_id TEXT NOT NULL, type TEXT NOT NULL CHECK(type IN ('decision','pattern','error','preference','context','tool_usage')), content TEXT NOT NULL, summary TEXT NOT NULL, confidence REAL NOT NULL DEFAULT 0.5, access_count INTEGER NOT NULL DEFAULT 0, created_at TEXT NOT NULL, last_accessed TEXT NOT NULL, FOREIGN KEY (project_id) REFERENCES projects(id)
  - `preferences` table: id TEXT PRIMARY KEY, key TEXT NOT NULL UNIQUE, value TEXT NOT NULL, confidence REAL NOT NULL DEFAULT 0.5, source_session TEXT, created_at TEXT NOT NULL, last_updated TEXT NOT NULL
  - `observations_fts` virtual table: `CREATE VIRTUAL TABLE IF NOT EXISTS observations_fts USING fts5(content, summary, content=observations, content_rowid=id)`
  - Three triggers: obs_ai (AFTER INSERT), obs_ad (AFTER DELETE), obs_au (AFTER UPDATE) — exactly as in 15-RESEARCH.md Pattern 2
  - Index: `CREATE INDEX IF NOT EXISTS idx_observations_project ON observations(project_id)`
  - Index: `CREATE INDEX IF NOT EXISTS idx_observations_type ON observations(type)`

Create `src/memory/repository.ts`:
- Import `getMemoryDb` from `"./database"`, schemas from `"./schemas"`, types from `"./types"`
- `insertObservation(obs: Omit<Observation, "id">, db?: Database): Observation` — validate with observationSchema, use db.run() INSERT, return with generated id from db.query("SELECT last_insert_rowid()"). All params as positional `?` bindings (not named).
- `searchObservations(query: string, projectId: string | null, limit?: number, db?: Database): Array<Observation & { ftsRank: number }>` — FTS5 MATCH query with BM25 rank, JOIN to observations table, filter by project_id, ORDER BY fts_rank, LIMIT (default 20). Return array of observation objects with ftsRank.
- `upsertProject(project: Project, db?: Database): void` — INSERT OR REPLACE into projects.
- `getProjectByPath(path: string, db?: Database): Project | null` — SELECT by path, return null if not found.
- `getObservationsByProject(projectId: string | null, limit?: number, db?: Database): readonly Observation[]` — SELECT ordered by created_at DESC, LIMIT (default 50).
- `upsertPreference(pref: Preference, db?: Database): void` — INSERT OR REPLACE into preferences.
- `getAllPreferences(db?: Database): readonly Preference[]` — SELECT all from preferences.
- `deleteObservation(id: number, db?: Database): void` — DELETE from observations WHERE id = ?.
- `updateAccessCount(id: number, db?: Database): void` — UPDATE observations SET access_count = access_count + 1, last_accessed = ? WHERE id = ?.
- All functions accept optional `db` parameter; default to `getMemoryDb()` if not provided (for testability with :memory: databases).
